## Regras de Linguagem

Liste combinados importantes para manter consistência na comunicação.

- Usar `termo oficial` em vez de `termo alternativo`.
- Evitar siglas sem descrição.
- Validar novos termos com negócio antes de adotar no código e na documentação.
